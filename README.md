# C_Quadratic_Equation
Задание: Создайте модуль quadratic_equation на языке C, содержащий реализацию функции solve_equation. Аргументы функции — 3 вещественных числа a, b, c. Функция решает квадратное уравнение ax^2+bx+c=0.  Модуль будет использоваться в качестве расширения для многофункциональной математической системы, над которой работает несколько десятков разработчиков.
